# u-music

> 一款基于Vue2.x开发的移动端音乐播放器

## 效果图展示

![image](https://github.com/bruceway17/u-music/blob/master/static/2017-11-26_all.png)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
